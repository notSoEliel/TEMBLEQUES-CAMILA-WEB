# Regla 12: Estética Visual Premium (Abandono del Neobrutalismo)

Esta regla define la identidad visual innegociable de Tembleques Camila, estableciendo la evolución desde un diseño experimental hacia uno de lujo y alta costura digital.

## 1. Declaración de Identidad
La estética del proyecto es **Premium, Elegante y Limpia**. El objetivo es evocar la calidad de las polleras y vestimentas folclóricas de Panamá a través de una interfaz moderna.

## 2. Abandono del Neobrutalismo [CRITICAL]
Queda estrictamente prohibido el uso de elementos del estilo Neobrutalista que se usaron en fases previas:
- **Prohibición de Bordes Negros Gruesos**: No se permiten bordes sólidos de 2px o más en contenedores, botones o tarjetas. Los bordes deben ser sutiles (`border-border/40`) o inexistentes si se usa sombra.
- **Adiós a los Colores Saturados Planos**: No usar colores primarios puros sin degradados sutiles o sin el refinamiento del formato OKLCH.

## 3. Principios de Diseño
- **Espaciado Generoso**: El contenido debe "respirar". Se deben usar paddings y margins amplios para evitar la saturación visual.
- **Jerarquía Visual**: Uso de tipografía Serif (`Playfair Display`) para títulos que transmitan tradición y Sans (`Inter`) para legibilidad en el cuerpo.
- **Micro-interacciones**: Cada acción del usuario debe tener una respuesta visual sutil (hover suave, cambio de opacidad, desplazamiento mínimo).

## 4. Estándar de Imágenes
- **No Placeholders**: Prohibido el uso de imágenes de relleno genéricas. Si no hay una imagen real, se debe usar un componente de "Estado Vacío" elegante o una ilustración generada que respete la paleta de la marca.
- **Tratamiento de Imagen**: Las imágenes de productos deben tener esquinas redondeadas coherentes con el sistema de diseño (`rounded-xl` o superior).

> [!IMPORTANT]
> El diseño debe sentirse como una boutique de lujo, no como una tienda de descuento o una aplicación técnica cruda.
