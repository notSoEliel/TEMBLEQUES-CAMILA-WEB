# Reglas del Panel Administrador - Tembleques Camila

El panel de administrador debe permitir gestionar completamente el inventario y el ciclo de vida de las reservas.

## 1. Dashboard Principal
Debe incluir métricas clave (KPIs):
- Reservas activas actualmente.
- Próximas devoluciones (fechas cercanas).
- Ingresos mensuales.
- Productos más alquilados.
- Incidencias por daño.

## 2. Gestión de Inventario (Productos)
- CRUD completo de productos.
- Edición y control de stock.
- Capacidad de marcar un producto como: `disponible` o `en mantenimiento`.
- Gestión de galería de imágenes por producto.

## 3. Gestión de Reservas
- Capacidad de aprobar reservas (si aplica el flujo manual).
- Actualizar el estado físico: marcar como `entregado`, `devuelto`, `atrasado`, o `dañado`.

## 4. Gestión de Usuarios
- Ver historial completo de alquileres de un cliente específico.
- Registro y seguimiento de incidencias o penalizaciones asociadas a un cliente.
