# TypeScript Exclusive Workspace

- We are strictly a TypeScript project.
- Always write new code, specially in the frontend, using TypeScript (`.ts`, `.tsx`).
- Do not use plain JavaScript (`.js`, `.jsx`) for new features.
- If asked to create a file or change the frontend, always output TypeScript code.
- Prepare the project to be fully migrated to TypeScript in the future.
